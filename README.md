## 💻 Installation

### Requirements

- Flutter SDK (>= 3.0.0)
- Dart SDK (>= 3.0.0)

### Steps

```bash
# Clone the project
git clone https://github.com/Chanomhub/ChanoLite.git
cd ChanoLite

# Install dependencies
flutter pub get

# Run the app
flutter run
```
